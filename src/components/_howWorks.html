EINZEL-HANDEL/ONLINE

  • KAUFVOR- GANG

    Ware wie gewohnt im Internet bestellen.

  • ABHOLSTATION AUSWÄHLEN

    Bei Eingabe der Lieferadresse ganz einfach die nächstgelegene MyLocker24-Box
    auswählen.

  • LIEFERUNG DER WARE ZUR ABHOLSTATION

    Zusteller liefern die Ware an die gewünschte Adresse und
    hinterlegen Sie in einem
    Schließfach.

  • CODE WIRD VERSANDT

    Wurde die Ware hinterlegt, erhalten Kunden automatisch eine Nachricht auf das Smartphone mit dem PIN zur Abholung.

  • ÖFFNEN UND WARE ENTNEHMEN

    Bei erfolgreicher PIN-Ein- gabe öffnet sich das
    Fach und die Ware kann
    entnommen werden.
